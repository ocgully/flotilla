# Small, fast notes

A notes CLI so trivial you don't think about it.

- **Fast.** Cold-start under 100 ms on a mid-range laptop.
- **Plain.** One file, one line per note. No database. No index. No daemon.
- **Obvious.** Four commands (`add`, `list`, `search`, `done`). You can
  learn the whole thing in 30 seconds.
- **Boring.** If you're wondering "what does this do in the weird case?",
  the answer is almost always "the naive thing."

The domain is deliberately boring because this project exists to showcase
the surrounding ecosystem — Mercator + Hopewell + Pedia + the AgentFactory
core-agent bundle — not a novel note-taking paradigm.
